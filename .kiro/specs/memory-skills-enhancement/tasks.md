# 实现计划：记忆系统与 Skills 热加载增强

## 概述

本实现计划将 design.md 中的设计转换为可执行的编码任务。实现顺序遵循依赖关系：先实现基础加载器，再实现监听器，最后实现高级功能（向量检索、事实提取）。
